## Prerequisites

Install the following using zypper

```bash
  sudo -E zypper install gcc-c++ \
gcc-fortran \
make \
libX11-devel \
blas-devel \
lapack-devel \
freeglut-devel \
m4 \
git
```

Download and install one our redistributable packages according to your version of OpenSUSE.

- OpenSuSE: !!package name arch=opensuse15!!
