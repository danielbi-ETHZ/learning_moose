# Alert Extension

!alert note
Note only.

!alert note title=Title
Note with a title.

!alert note title=Title prefix=False
Note with a title but without prefix.

!alert error
An error.

!alert warning
A warning.

!alert construction
Under construction.

!alert construction icon=False
Under construction.


!alert! error
An alert with some code.

```c++
int x;
```
!alert-end!
